THE GAME

O GAME

PUT THE PORTALS é um jogo da categoria puzzle, onde o jogador controlará o Bibbo, personagem principal do game que usar seus poderes para resolzer os mais variados desafios e enfrentar uma gama inimigos. Ele usa seus portais para se locomover.
